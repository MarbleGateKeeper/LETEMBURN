# LET!THEM!BURN!
An addon for Create: Simulated Series.

***Make those technical miracles to happen in correct place.***

Current function:
* Draconic Evolution: *Mega Explosion* will always happen in right place when on Simulated Contraption. Fix rendering of Reactor Core on Simulated Contraption.
* Mechanism: If a Cardborad Box packaged an Exploding Draconic Reactor Core and gets hit, it will break and release that explosion instantly. Here we go nuke missile!!
